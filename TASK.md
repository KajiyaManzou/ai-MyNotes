# ai-MyNotes タスクリスト

## プロジェクト概要
- **プロジェクト名**: ai-MyNotes
- **目的**: 自分専用のシンプルで高速なメモアプリケーション開発
- **技術スタック**: Flutter (Dart) + SQLite
